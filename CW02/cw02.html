Excercise 6

Since all pages need to link to the school homepage. A theme for the home page needs to be determined first, components including the theme color(could be the taken from school badge).
This can be acheieved by creating a .background class Also, the school badge needs to be be displayed using either span or div elements paired with a pseudo-class.

.theme_color {theme color throughout website}
.badge {background image: url(school-badge.gif)}
<#title> tags need to be standarized to make navigating through website easier
<#head> tag pared with a .school_name{fonts, colors, location} class to display school name on main sites
A .dept_name{fonts, colors, location} inherented from school_name to display department names on department sites, essential changes can be made
Multiple pseudo-class point to <#table> tag to achieve calendar, timetable
Use :hover{color:***} link state to standarize all the hover action throughout website
A <#footer> tag to dispay info like address, phone num, etc.
Multiple <#a href=url> links to school/departments social media
The body{margin: ; padding: ; etc.} style sheet can provide a basic layout for most pages
